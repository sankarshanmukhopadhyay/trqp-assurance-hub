# Ecosystem Onboarding Use Case

## Business problem

An ecosystem operator needs to decide whether a registry may be admitted as a source of claims, membership status, or authority.

## Assurance decision

The operator is not merely asking whether the registry endpoint responds. The operator is deciding whether the registry can participate in the ecosystem control plane.

## Evidence package

- CTS scenario profile report for ecosystem onboarding.
- TSPP relying-party/publication controls.
- Combined Assurance Manifest.
- Public Assurance Summary.
- Limitations and redress metadata.

## Consumer impact

Consumers benefit because the decision to rely on a registry becomes evidence-bound, freshness-aware, and contestable.
