| Assurance Hub | Conformance Suite | TSPP | Status | Notes |
|---|---|---|---|---|
| v1.1.0 | v0.9.1 | v0.7.1 | supported | Current integration baseline with trust registry service and machine-readable assurance profiles |
| v1.1.x | v0.9.x | v0.7.x | supported | Operational Stack baseline |
| v1.1.x | v0.8.x | v0.6.x | legacy | Works for older combined-assurance flows but lacks shared `run_id` and `target_id` fields |
