# TRQP Business Use Cases

These use cases translate the Operational Trust Stack into business decisions. Each use case identifies the decision being made, the evidence required, the appropriate assurance level, and the consumer or relying-party impact.

| Use case | Primary decision | Recommended baseline |
|---|---|---|
| Ecosystem registry onboarding | Admit a registry into an ecosystem control plane. | AL2, AL3 where high reliance applies. |
| Public-sector authoritative directory | Accept an authoritative directory as a public reference point. | AL3. |
| Supply-chain participant registry | Accept participant status or capability claims. | AL2 to AL3. |
| AI agent service registry | Accept service, authority, or delegation metadata for agentic systems. | AL3. |
| Content authenticity verifier registry | Accept verifier registry claims for content authenticity workflows. | AL2 to AL3. |
