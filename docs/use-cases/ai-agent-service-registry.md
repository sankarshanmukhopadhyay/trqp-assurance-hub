# AI Agent Service Registry

## Business problem

An ecosystem verifies agent service metadata, delegated authority, and operational posture before allowing automated actors to rely on registry claims.

## Who needs assurance?

- Ecosystem operator.
- Registry operator.
- Relying party.
- Assessor or auditor.
- Affected user or consumer.

## Evidence required

- CTS scenario profile report.
- TSPP posture report with relying-party controls.
- Combined Assurance Manifest.
- Public Assurance Summary.
- Limitations and contact/redress metadata.

## Adoption impact

This use case converts trust from a narrative claim into an executable onboarding and review process.
