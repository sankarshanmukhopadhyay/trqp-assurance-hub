# v0.5.0 — GRID readiness kernel

This release adds a minimal, verifier-first kernel for **Global Registrar Information Directory (GRID)** style deployments, to demonstrate how TRQP can be extended across different trust registry/directory implementations.

## Added
- GRID profile (Markdown + machine-readable JSON)
- Registrar listing schema + example
- GRID status feed schema + example
- Assurance eligibility mapping for directory listings
- Minimal threat annex + verifier-first verification workflow
- Crosswalk guidance referencing UN/CEFACT GTR/GRID and EBSI registry APIs

## Fixed
- Converted `tools/control-catalog.json` into valid JSON so example validation can run cleanly.

## References
- UN/CEFACT GTR / GRID: https://un.opensource.unicc.org/unece/uncefact/gtr/
- EBSI Trusted Issuers Registry: https://hub.ebsi.eu/apis/pilot/trusted-issuers-registry
