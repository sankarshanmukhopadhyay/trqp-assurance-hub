# TRQP Assurance Hub v0.5.0

Release date: 2026-03-02

## What changed

This release adds the minimum viable kernel to support a **GRID-style Global Registrar Information Directory**:
- Machine-readable registrar listing schema (`schemas/registrar.schema.json`)
- Signed status feed schema (`schemas/grid-status-feed.schema.json`)
- GRID profile + assurance-to-directory eligibility mapping
- Verifier-first guidance and a minimal threat annex
- Compatibility notes for GTR interop

## Why it matters

This turns GRID from an idea into an auditable directory surface: third parties can validate listing integrity, lifecycle status, and assurance posture without private context.
