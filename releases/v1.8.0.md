# TRQP Assurance Hub v1.8.0 Release Notes

## Summary

v1.8.0 is the Hub portion of the Operational Trust Stack Maturity Release. It makes the Hub the canonical adoption front door for the coordinated CTS, TSPP, and Hub repository set.

## Highlights

- Added release governance that distinguishes patch, minor, maturity, and no-release changes.
- Added change-intake guidance so maintainers can batch small changes and prioritize high-value commits.
- Added release validation evidence for the coordinated Hub v1.8.0 / CTS v1.5.0 / TSPP v0.13.0 tuple.
- Refreshed README and roadmap content to explain the adoption path from protocol conformance to posture evidence to public assurance publication.
- Added GitHub templates to require evidence impact, validation impact, and compatibility review.
