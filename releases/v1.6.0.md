# v1.6.0 — Operational Trust Stack: Public Assurance and Adoption Readiness

## Release purpose

This release moves the Hub from internal assurance orchestration to public assurance publication. It adds Public Assurance Summary generation, adoption packs, business use-case guides, consumer impact documentation, and coordinated compatibility with CTS v1.3.0 and TSPP v0.11.0.

## Highlights

- Public Assurance Summary schema, generator, validator, and examples.
- Relying-party assurance summary example.
- Business use-case library and adoption kit.
- Refreshed compatibility matrix for the coordinated release train.
- Documentation cleanup for stale release references.
