# v1.1.0 — Operational Stack Baseline

This release turns the Assurance Hub into the practical front door for the TRQP triad.

## Highlights

- Add `docs/guides/run-the-stack.md` as the canonical end-to-end operator guide.
- Add `tools/run_operational_stack.py` to build a portable stack artifact directory from CTS and TSPP reports.
- Extend `tools/generate-manifest.py` and the combined-assurance manifest schema with shared `run_id` and `target_id` support.
- Add a compatibility matrix and a GitHub Actions smoke workflow for the integrated stack example.
