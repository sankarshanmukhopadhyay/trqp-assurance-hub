# Operational Stack example

This example shows the minimum artifact set required for the Operational Stack baseline.

## Layout

- `demo-input/cts-report.json` — sample CTS machine-readable report
- `demo-input/tspp-report.json` — sample TSPP posture report
- `expected-output/combined-assurance-manifest.json` — expected roll-up manifest
- `../../profiles/al2-verified.yaml` — example machine-readable assurance profile for publication

These files are intentionally small. The point is to make the integrated shape obvious, not to cosplay a full production audit.
