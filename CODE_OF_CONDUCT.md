# Code of Conduct

We aim for a collaborative, professional environment.

## Expected behavior

- Be respectful and constructive
- Assume good intent; ask clarifying questions
- Focus on evidence and reproducible claims

## Unacceptable behavior

- Harassment or discriminatory language
- Personal attacks or sustained hostility
- Publishing private information without consent

## Reporting

If something goes sideways, open an issue in this repo describing the incident at a high level. If the report must be private, contact the maintainer listed in the repository metadata.
